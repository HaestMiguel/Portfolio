Decision matrix

Research & analysis fase

Context en achtergrond

Tijdens mijn stage heb ik een BI decision matrix gemaakt. Dit houdt in dat ik een onderzoek heb gedaan naar 5 BI tools: PowerBI, Tableau, Qlik Sense, AWS Quicksight en Google Data Studio. Hiervoor heb ik een template gekregen dat ik ruwweg moest volgen. Hierbij vergeleek ik onderstaande delen van de BI tool.

  • Data connectivity
  • Data preperation & modeling
  • Interactive dashboards & exploration
  • Sharing & publishing
  • Scalability & storage
  • Administration, security & monitoring
  • Overall platform
  • Pricing

Ik documenteerde dit op 3 verschillende manieren:

  • Eerst in een decision matrix met volzinnen hierbij kan je specifieke details per tool terugvinden. Hier krijg je de meeste informatie over elke item.
  • Daarna in een decision matrix zodat je de verschillende tools sneller met elkaar kan vergelijken
  • Tot slot werd er op punten gescoord zodat je de verschillende tools rekenkundig kan vergelijken.

Conclusie

Nodige skills

Voor dit deel van mijn stage moest ik weten hoe:

  • Ik informatie opzocht
  • Ik informatie verwerkte in een matrix
  • Ik kritisch nadacht over wat er over een tool wordt geschreven

Wat heb ik geleerd van dit project

Tijdens dit deel van mijn stage leerde ik veel bij over de BI tools en wat de beste manier situatie is om de BI tools te gebruiken. Bij dit deel van het project was het vooral theoretisch. Ik reflecteerde over dit deel met mijn stagementor.

Dit was misschien niet het leukste moment van mijn stage maar het was zeker een van de meest leerrijke.

  • Opdracht: Decision matrix
  • Fase: Analyse fase
  • Opgelevert werk: BI Decision Matrix
